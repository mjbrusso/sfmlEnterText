# sfmlEnterText
Sample code for entering text in a C++/[SFML](https://www.sfml-dev.org/) game.
## Build

```c++
g++ entertext.cpp  -o entertext -lsfml-graphics -lsfml-window -lsfml-system
```

## Run
```bash
./entertext
```
